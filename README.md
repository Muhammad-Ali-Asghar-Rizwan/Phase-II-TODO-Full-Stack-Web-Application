# Todo App (Phase 2)

## Setup

### Backend
1. Navigate to `backend/`
2. Install dependencies: `uv sync`
3. Run: `uv run fastapi dev src/main.py`

### Frontend
1. Navigate to `frontend/`
2. Install dependencies: `npm install`
3. Run: `npm run dev`

## Configuration
Copy `.env.example` to `backend/.env` and `frontend/.env` (or set env vars appropriately).
